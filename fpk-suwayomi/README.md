# Suwayomi Server

Suwayomi 是一个开源的漫画阅读服务器，支持多种漫画源，提供 Web 阅读界面和移动端应用。

## 功能特性

- **多源支持**: 支持多种漫画网站和源
- **在线阅读**: 提供 Web 阅读界面
- **移动应用**: 支持 Android/iOS 客户端
- **书架管理**: 支持收藏和分类管理
- **下载功能**: 支持漫画下载到本地
- **阅读记录**: 保存阅读进度和历史

## 安装要求

- 系统架构: x86_64
- 运行环境: Linux
- 端口: 14567
- 内存: 建议至少 1GB RAM
- 存储: 建议至少 10GB 可用空间

## 使用方法

1. 安装应用后，通过浏览器访问 `http://<你的NAS IP>:14567`
2. 添加漫画源扩展
3. 搜索和浏览漫画
4. 添加到书架开始阅读
5. 配置下载和阅读设置

## 客户端支持

- **Web 客户端**: 通过浏览器直接访问
- **Android 应用**: Tachiyomi 应用 + Suwayomi 插件
- **iOS 应用**: 支持通过 TestFlight 安装
- **桌面客户端**: 支持桌面浏览器访问

## 扩展管理

- **源扩展**: 安装各种漫画源扩展
- **主题扩展**: 安装界面主题
- **工具扩展**: 安装辅助工具扩展
- **备份扩展**: 支持数据备份和恢复

## 注意事项

- 漫画源可能随时失效，需要定期更新
- 下载漫画时注意版权和法律法规
- 建议定期备份书架数据
- 关注官方更新，及时升级版本
