# AllinSSL

AllinSSL 是一个集证书申请、管理、部署和监控于一体的 SSL 证书全生命周期管理工具。

## 功能特性

- **证书申请**: 支持 Let's Encrypt 证书自动申请
- **多平台部署**: 支持 CDN、面板、云存储等多种平台
- **证书监控**: 实时监控证书过期状态
- **安全入口**: 提供安全入口保护功能
- **批量管理**: 支持批量证书管理和部署

## 安装要求

- 系统架构: x86_64
- 运行环境: Linux
- 端口: 17979

## 使用方法

1. 安装应用后，通过浏览器访问 `http://<你的NAS IP>:17979`
2. 按照向导完成初始配置
3. 添加域名并申请 SSL 证书
4. 配置证书部署到目标平台

## 注意事项

- 本应用为第三方封装版本，基于开源项目构建
- 请遵守相关法律法规，合法使用 SSL 证书管理功能
- 建议定期备份证书和配置数据
