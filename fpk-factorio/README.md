# 异星工厂服务器 (Factorio Server)

异星工厂（Factorio）服务器 - 一个自动化工厂建设游戏服务器。

## 功能特性

- **多人联机**: 支持多人同时在线游戏
- **模组支持**: 支持游戏模组加载和管理
- **存档管理**: 支持游戏存档的备份和恢复
- **性能监控**: 实时监控服务器性能和资源使用
- **配置管理**: 支持游戏配置文件的在线编辑

## 安装要求

- 系统架构: x86_64
- 运行环境: Linux
- 端口: 34197 (游戏端口)
- 内存: 建议至少 2GB RAM
- 存储: 建议至少 10GB 可用空间

## 使用方法

1. 安装应用后，通过浏览器访问 `http://<你的NAS IP>:34197`
2. 创建新的游戏世界或加载现有存档
3. 配置游戏设置和模组
4. 分享服务器地址给其他玩家

## 游戏配置

- **游戏模式**: 自由模式、战役模式等
- **地图设置**: 可自定义地图大小、资源分布等
- **难度设置**: 可调整敌人强度、研究速度等
- **模组管理**: 支持 Steam 创意工坊模组

## 注意事项

- 游戏需要正版 Factorio 客户端
- 建议为服务器分配足够的 CPU 和内存资源
- 定期备份游戏存档以防数据丢失
- 关注官方更新，及时升级服务器版本
