# 夸克网盘自动转存

夸克网盘自动转存工具，集成 pansou 版本，实现一键搜索转存。

## 功能特性

- **自动转存**: 支持自动转存搜索到的资源
- **批量操作**: 支持批量转存和管理
- **任务管理**: 提供任务队列和进度监控
- **搜索集成**: 集成 pansou 搜索功能
- **历史记录**: 保存转存历史记录

## 安装要求

- 系统架构: x86_64
- 运行环境: Linux
- 端口: 15005
- 依赖: 需要夸克网盘账号

## 使用方法

1. 安装应用后，通过浏览器访问 `http://<你的NAS IP>:15005`
2. 登录夸克网盘账号
3. 配置自动转存规则
4. 使用搜索功能查找资源
5. 开始自动转存任务

## 主要功能

- **资源搜索**: 支持多种资源类型搜索
- **智能过滤**: 自动过滤无效和重复资源
- **速度优化**: 多线程转存提升速度
- **错误重试**: 自动重试失败的任务
- **通知提醒**: 任务完成通知

## 注意事项

- 需要有效的夸克网盘账号
- 遵守网盘服务条款和使用规范
- 注意转存频率，避免账号被封
- 定期清理已完成的任务记录
