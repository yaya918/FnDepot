# iCloud-Photos

一个带有Web界面的iCloud照片下载工具，基于icloudpd-web项目。支持Linux系统，提供直观的Web界面进行配置和管理，支持多账号、定时同步、实时监控等功能。

## 功能特性

- **Web界面管理**: 提供直观的Web界面进行配置和管理
- **多账号支持**: 支持多个iCloud账号同时管理
- **定时同步**: 支持定时自动同步照片和视频
- **实时监控**: 实时监控同步进度和状态
- **自动Cookie刷新**: 自动刷新iCloud登录Cookie，保持会话有效
- **按相册分类**: 支持按相册分类存储下载的文件
- **增量下载**: 支持增量下载，只下载新增内容
- **多平台支持**: 支持Linux系统，专为飞牛NAS优化

## 安装要求

- 系统架构: x86_64
- 运行环境: Linux
- 端口: 18088
- 依赖: Python 3.12

## 使用方法

1. 安装应用后，通过浏览器访问 `http://<你的NAS IP>:18088`
2. 首次使用时需要配置iCloud账号信息（passToken和userId）
3. 在Web界面中设置同步选项和计划任务
4. 启动同步任务，开始下载照片和视频
5. 下载的文件默认保存在 `/vol【x】/@appshare/icloud-photos` 目录

## 配置说明

- **passToken**: iCloud账号的认证令牌
- **userId**: iCloud账号的用户ID
- **保存路径**: 默认为 `/vol【x】/@appshare/icloud-photos`，可在计划任务中修改
- **同步间隔**: 支持自定义同步时间间隔
- **文件过滤**: 支持按日期、类型等条件过滤文件

## 注意事项

- 确保网络可访问，能够连接iCloud服务器
- 首次使用需要获取有效的passToken和userId
- 保存路径中不能包含中文字符
- 建议定期检查同步状态和日志
- 确保有足够的存储空间保存照片和视频
- 如需更改保存路径，请在飞牛系统设置中添加相关文件夹权限
